## üìä Project Overview

**Project Name:** African Health Excellence Organisation (HEOSA)  
**Type:** Full-stack web application  
**Purpose:** Healthcare awards management, medical events, and community engagement

### Technology Stack

**Frontend:**
- React 18.3.1
- TypeScript 5.7.2
- Tailwind CSS 3.4.1
- React Router 6.22.3
- Vite 5.4.2

**Backend:**
- PHP 7.4+
- PHPMailer 6.8
- REST API Architecture

**Build & Deploy:**
- Vite (Build tool)
- Composer (PHP dependencies)
- npm (JavaScript dependencies)

## üéØ Core Features

1. **Awards System** - Comprehensive awards management with voting
2. **Medical Events** - Event listings and registrations
3. **Gallery System** - Multi-year photo and video galleries
4. **Nomination System** - Email-based nomination workflow
5. **Voting Platform** - SMS, WhatsApp, and email voting
